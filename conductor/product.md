# Conceito Inicial
O projeto é um sistema de gerenciamento de tarefas focado em produtividade chamado "Flow State", projetado para ajudar os usuários a gerenciar seu fluxo de trabalho, rastrear métricas e orquestrar demandas usando React e Supabase.

# Definição do Produto

## Público-Alvo
- Indivíduos de alto desempenho que buscam otimizar seu "trabalho profundo" (deep work) ou estado de fluxo pessoal.
- Usuários que buscam uma solução de usuário único, auto-hospedada (via Docker) com integração simples com o Supabase.

## Objetivos Principais
- **Eliminar a Ilusão de Produtividade:** Combater a dispersão de esforço e a falta de rastreamento que leva à sensação de estar ocupado sem ser produtivo.
- **Maximizar o Foco:** Minimizar distrações e permitir períodos sustentados de "trabalho profundo".
- **Métricas Acionáveis:** Fornecer cálculo preciso de horas de trabalho/foco para impulsionar resultados extraordinários através de alta produtividade e velocidade de execução.
- **Priorização Clara:** Oferecer uma visão clara da carga de trabalho e das prioridades diárias.

## Funcionalidades Principais
- **Painel em Tempo Real:** Uma visão centralizada das tarefas ativas e métricas de foco.
- **Cronômetro de Foco Integrado:** Um temporizador embutido para rastrear sessões de trabalho profundo e reforçar blocos de foco.
- **Orquestrador de Demandas:** Um sistema para criar, categorizar e priorizar rapidamente demandas/tarefas.
- **Implantação Conteinerizada:** Configuração completa de Docker e Docker Compose para fácil implantação em VPS/Portainer.
- **Configuração Centralizada:** Gerenciamento seguro das chaves de API do Supabase e Google AI (Gemini) via arquivos `.env`.
- **Pronto para CI/CD:** Configuração para enviar imagens para o Docker Hub e código para o GitHub.
