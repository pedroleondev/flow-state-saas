# Pilha Tecnológica (Tech Stack)

## Frontend
- **Linguagem:** [TypeScript](https://www.typescriptlang.org/) - Para garantir segurança de tipos e melhor manutenção do código.
- **Framework:** [React 19](https://react.dev/) - Biblioteca principal para construção da interface do usuário.
- **Ferramenta de Build:** [Vite](https://vitejs.dev/) - Para um ambiente de desenvolvimento rápido e builds otimizados.
- **Estilização:** [Tailwind CSS](https://tailwindcss.com/) - Framework CSS utilitário para design rápido e responsivo.
- **Ícones:** [Lucide React](https://lucide.dev/) - Conjunto de ícones consistentes e leves.
- **Roteamento:** [React Router DOM](https://reactrouter.com/) - Gerenciamento de navegação e rotas da aplicação.

## Backend e Serviços
- **Backend as a Service (BaaS):** [Supabase](https://supabase.com/) - Utilizado para autenticação, banco de dados PostgreSQL em tempo real e armazenamento.
- **Inteligência Artificial:** [Google Gemini API](https://ai.google.dev/) - Integrada para orquestração de demandas e funcionalidades inteligentes.

## Infraestrutura e Deployment
- **Conteinerização:** [Docker](https://www.docker.com/) & [Docker Compose](https://docs.docker.com/compose/) - Para padronização do ambiente e facilidade de implantação.
- **Hospedagem:** VPS com [Portainer](https://www.portainer.io/) para gerenciamento de containers.
- **Gerenciamento de Segredos:** Variáveis de ambiente via arquivos `.env`.
